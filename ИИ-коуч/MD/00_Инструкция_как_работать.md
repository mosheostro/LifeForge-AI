# КАК РАБОТАТЬ С ИИ-КОУЧЕМ

Это система персонального ИИ-коуча. Не «спроси у ChatGPT» — а полноценный долгосрочный коуч, который знает вас, помнит ваши паттерны и работает по проверенным методикам.

---

## Архитектура системы

Три слоя:

**Слой 1 — Контекст обо мне.** Кто я, какие у меня цели, ценности, проекты. Загружается один раз, дополняется по мере роста.

**Слой 2 — Методики коучинга.** 4 модуля: ICF, GROW, Колесо баланса, Co-Active. Каждый — для своего типа запросов. Можно комбинировать.

**Слой 3 — Ритмы.** 4 формата сессий: Утро, Вечер, Неделя, Месяц. Каждый — отдельный сценарий с конкретными вопросами и финальным артефактом.

## Где собирать (3 варианта)

### Вариант 1: Claude Projects (рекомендую)

1. Создайте проект «Мой коуч»
2. В Project Instructions вставьте целиком содержимое **01_Системный промпт**
3. В Knowledge загрузите 8 файлов (методики + ритмы)
4. Отдельно подготовьте свой файл «Контекст обо мне» (см. ниже) и тоже в Knowledge
5. Открывайте новый чат — пишите запрос — работайте

### Вариант 2: NotebookLM

1. Создайте блокнот «Коуч»
2. Загрузите все 9 файлов (системный промпт + 8 модулей) как источники

4. В строке запроса начинайте: «Действуй по правилам из 01_Системный промпт. Запрос: …»

## Что положить в «Контекст обо мне»

Создайте один документ с этой структурой. Это ваш личный файл, в проект не входит.

**Минимум:**
- Биография (1-2 страницы)
- Текущие проекты и роли
- Ключевые цели на год
- Ваши главные ценности (5-7 пунктов)

**Желательно:**
- Распаковка личности (Gallup, MBTI, тесты)
- История важных решений и их результатов
- Ключевые люди и роли в вашей жизни
- Финансовая картина крупными мазками

**Идеально:**
- Дневник инсайтов за последние месяцы
- Список «что меня наполняет / что опустошает»
- Принципы, которыми руководствуетесь
- Что больше не делаете и почему

Чем плотнее контекст — тем глубже работает коуч. Но даже минимума достаточно для старта.

## Как использовать в работе

### Свободная сессия

Просто пишете: *«С чем сегодня?»* или *«Помоги разобрать вопрос — …»*. Коуч сам выберет подходящую методику исходя из запроса.

---

### Шпаргалка команд

Скопируйте и сохраните — это все рабочие команды системы.

**Запуск ритмов (структурированные сессии):**

| Команда | Что запускает | Длительность |
|---|---|---|
| `Утренний разбор` | Ритм Утра — фокус дня и первый шаг | 5-7 мин |
| `Вечернее ретро` | Ритм Вечера — итог дня и инсайт | 10 мин |
| `Еженедельный обзор` | Ритм Недели — итог + план через GROW | 30 мин |
| `Месячный итог` | Ритм Месяца — глубокий аудит | 60-90 мин |

**Запуск методик (свободные сессии):**

| Команда | Что запускает | Когда применять |
|---|---|---|
| `Давай по ICF` | Чистый коучинг через открытые вопросы | Прояснение, осознание |
| `Разбираем по GROW` | Структурный разбор задачи | Конкретная задача / решение |
| `Время для колеса баланса` | Аудит сфер жизни | Обзор жизни целиком |
| `Запускаем Co-Active` | Работа с ценностями и смыслом | Большие решения, поиск смысла |

**Комбинации (самое мощное):**

| Команда | Что произойдёт |
|---|---|
| `Сделай колесо баланса, потом GROW по самой низкой сфере` | Аудит → действие |
| `Сначала ICF — прояснить, потом GROW — что делать` | Прояснение → структура |
| `ICF по этому решению, в конце — 3 варианта по GROW` | Осознанность → варианты |
| `Месячный итог + Co-Active в блоке про большую картину` | Рутина + глубина |

**Свободные команды:**

| Команда | Что произойдёт |
|---|---|
| `С чем сегодня?` | Коуч сам подберёт методику |
| `Хочу разобрать вопрос: [запрос]` | Свободная сессия по запросу |
| `Покажи доступные ритмы` | Коуч перечислит варианты |
| `Покажи доступные методики` | Коуч перечислит варианты |

### Логика выбора подхода

Если не знаете, что выбрать — ориентируйтесь так:

- **Утро началось** → `Утренний разбор`
- **День заканчивается** → `Вечернее ретро`
- **Воскресенье** → `Еженедельный обзор`
- **Последний день месяца** → `Месячный итог`
- **Конкретная задача в течение дня** → `Разбираем по GROW`
- **«Что-то не так, но непонятно что»** → `Давай по ICF`
- **«Внутри пусто, ради чего всё это»** → `Запускаем Co-Active`
- **«Хочу честно посмотреть на всю жизнь»** → `Время для колеса баланса`

## С чего начать в первую неделю

**День 1.** Соберите Project, загрузите минимальный контекст, запустите главный промпт. Проведите первую утреннюю сессию.

**Неделя 1.** Каждое утро и вечер — короткие сессии. Воскресенье — первый еженедельный обзор.

**Месяц 1.** В конце месяца — первый месячный итог. Оцените, что работает. Доработайте контекст.

**Месяцы 2-3 и далее.** Стабильная практика. По мере накопления сессий коуч начинает видеть ваши повторяющиеся паттерны и темы, к которым вы возвращаетесь. При регулярной работе ИИ-коуч становится глубоким зеркалом — не благодаря «магии», а потому что у него накопилось много данных о вас и ваших циклах.

Темп индивидуален. Кому-то заметные результаты приходят через месяц, кому-то через три. Главное — регулярность, а не интенсивность.

## Антипаттерны

**Открывать каждый раз новый чат.** Теряется контекст сессий. Работайте в одном долгоиграющем чате (или нескольких — отдельный для ритмов, отдельный для глубоких разборов).

**Загрузить слишком много сырых файлов.** Лучше один структурированный документ «Контекст обо мне», чем 20 разрозненных.

**Просить совет вместо вопроса.** Коуч даст совет, если попросите. Но это не его роль. Сильнее — *«помоги мне самому увидеть»*, чем *«скажи, что делать»*.

**Пропускать ритмы дольше недели.** Утро-вечер можно иногда пропустить. Воскресный обзор и месячный итог — святое. Без них система разваливается.

**Превращать сессии в жалобу без действия.** Каждая сессия должна заканчиваться конкретным шагом. Без шага — это не коучинг, это разговор.

**Менять контекст вслед за настроением.** Если переписали ценности через неделю, потом ещё через две — система потеряет глубину. Меняйте контекст редко и осознанно.

## Финал

Коуч — это зеркало, не учитель. Он не знает, как вам жить. Но он помогает услышать самих себя без шума.

Используйте ежедневно — увидите перемены через месяц.
